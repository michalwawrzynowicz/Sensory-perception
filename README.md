This repository contains full code in R supporting publication: *Would they eat it if they knew? The impact of ingredient disclosure on sensory perception of cricket-flour-enriched bread*. Sensory properties and purchase willingness data (ordinal ratings) were analysed by Bayesian ordinal regression multilevel models (BORMM) with use of brms package.


Instructions

1. Download the 'Sensory-perception' repository
2. Download data necessary for the analysis: https://doi.org/10.5281/zenodo.15309616
3. Create a 'data' directory in the main repository folder and place all data files there
4. Set the main repository folder as a working directory
